Plugin Oficial readBlog - Haz que tu blog esté disponible en readblog con este plugin

== Description ==

readBlog es una aplicación móvil disponible en Google Play y App Store donde el usuario puede leer y navegar por tu blog Wordpress desde cualquier sitio, el plugin genera un código QR que deberás colocar en un sitio visible de tu web o compartirlo en redes sociales, foros, etc.

Cuando el usuario tiene la aplicación instalada en su dispositivo puede añadir el blog a la aplicación con solo escanear el código QR, el sistema agregará tu blog a su móvil y ya podrá navegar y leer todos los post que publiques esté donde esté.

Puedes descargar desde los markets a tu dispositivo móvil desde los siguientes enlaces

Descarga la app Android [Ir a Play Store](https://play.google.com/store/apps/details?id=com.taxarpro.readblog)
Descarga la app iOS [Ir a App Store](https://appsto.re/es/7EBKeb.i)

Notificaciones Push, a partir de la versión 1.0.5 podrás enviar notificaciones push a tus lectores desde tu Wordpress sin necesidad de configurar nada, podrás compartir un post o un mensaje con tus lectores en segundos y les llegará a todos una notificación al instante.

Tambien a partir de la versión 1.0.5 podrás ver el uso con Google Analytics, tienes dos opciones, puedes crear una vista nueva dentro de tu proyecto Google o puedes crear un nuevo codigo de seguimiento, lo recomendable es una nueva vista.

¿Echas de menos poder ingresar dinero si tus visitantes no están en tu blog? No pasa nada, puedes añadir el codigo de Admob en la pantalla de configuración del plugin, esto hará que readBlog muestre anuncios que va directamente a tu cuenta.

== Installation ==

Aquí se detalla como instalar y poner en funcionamiento el sistema

e.g.

1. Instalar el plugin desde wordpress.org/plugins o también puedes descargar el plugin y subirlo directamente al directorio /wp-content/plugins/.
2. Activar el plugin
3. En el menú lateral se ha creado un nuevo botón para acceder a la configuración del plugin, tienes que elegir el color que quieres que la aplicación tenga al navegar por tu blog cuando lo hayas seleccionado pulsa en guardar y guarda la imagen del código QR para compartirla o ponerla en tu blog.


== Frequently Asked Questions ==

= Es Gratis =
Tanto el plugin como la aplicación son completamente gratuitas, sin coste para ti ni tampoco para el usuario de la app.
= Solo quiero mostrar algunas categorias de mi web = 
No pasa nada, selecciona las categorías que quieres mostrar en readBlog, si ninguna de las categorias es seleccionada se seleccionarán todas.
= No puedo mandar notificaciones con readBlog =

Si no puedes mandar notificaciones desde readBlog, desactiva el plugin de readBlog y vuelve a activarlo, eso solucionará el problema.
